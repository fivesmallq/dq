# Usage
To add a dependency on dq using Maven, use the following:

```xml
<dependency>
    <groupId>im.nll.data</groupId>
    <artifactId>dq</artifactId>
    <version>0.9.7</version>
</dependency>
```

To add a dependency using Gradle:

```
dependencies {
  compile 'im.nll.data:dq:0.9.7'
}
```
